- page  request  session application  이 내장객체들에
    
    모두 같은 cnt 라는 key에 값을 담았을때,
    
    **내장객체에도 우선순위가 있기 때문에 page로 나타날것**
    
    이 경우 만약 request에 있는 key 값을 꺼내오고 싶을때

     **Scope 라는 한정사 를 붙이므로써** 그에 맞는 내장객체를 찾아서 값을 꺼내온다
